# Posea-Pose-Suggestion-App.
Awarded Top 10 Project designation among 33 competing final-year academic engineering projects.

Lumina is an intelligent, real-time photography assistant application engineered to bridge the gap between amateur photography and professional aesthetics. Powered by a Custom Multi-Label Convolutional Neural Network (CNN) and high-dimensional Vector Space Search, Lumina analyzes a live camera background to instantly extract environmental "DNA" and recommend gender-filtered, contextually perfect professional poses.

**Project Components:**

* 📱 **[Mobile App](https://github.com/supunjayamal/posea_mobile_app):** Built with Dart/Flutter for the cross-platform user interface.
* ⚙️ **[Backend](https://github.com/supunjayamal/posea_backend):** Python backend to handle requests, process data, and route information.
* 🧠 **[Machine Learning Model](https://github.com/supunjayamal/posea_background_classification_model):** Python-based background classification model used for pose analysis.
