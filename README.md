# Posea-Pose-Suggestion-App.
A full-stack application that analyzes and suggests poses using machine learning.

**Project Components:**

* 📱 **[Mobile App](https://github.com/supunjayamal/posea_mobile_app):** Built with Dart/Flutter for the cross-platform user interface.
* ⚙️ **[Backend](https://github.com/supunjayamal/posea_backend):** Python backend to handle requests, process data, and route information.
* 🧠 **[Machine Learning Model](https://github.com/supunjayamal/posea_background_classification_model):** Python-based background classification model used for pose analysis.
